# Quiz App
This app evaluates your knowledge on the topic of the quiz form the field of IT.

## Features
- Start the quiz
- Answer questions
- View your score

## Technologies
- React
- Vite
- Tailwind CSS

## Getting Started
1. Clone the repository
2. Install dependencies
3. Run the app
